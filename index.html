Proceso de instalación de mkdocs

Paso 1: Requisitos previos

  • Lo primero que hay que tener instalado es python (en mi caso Python3)
python --version

.

Paso 2: Instalación de MkDocs

  1. Abrir la terminal.
  2. Ejecutar el siguiente comando para instalar MkDocs a través de pip, el administrador de paquetes de Python:
pip install mkdocs

Este comando descargará e instalará MkDocs y sus dependencias.

Paso 3: Verificación de la instalación 1. Una vez completada la instalación, se puede verificar si MkDocs se ha instalado correctamente ejecutando este comando:

mkdocs --version

Esto mostrará la versión de MkDocs instalada.

Paso 4: Creación del proyecto de MkDocs 1. Dirigirse a la carpeta donde quier crear el proyecto. 2. Ejecutar el siguiente comando para crear el proyecto:

mkdocs new mkdocs-rafa

Esto crea una nueva carpeta llamada "mkdocs-rafa" que tiene los archivos para la documentación.

Paso 5: Ejecución del servidor de desarrollo

  1. Cambia al directorio del proyecto de documentación:
cd mkdocs-rafa
  1. Ejecutae este comando para iniciar el servidor de desarrollo de MkDocs:
mkdocs serve
